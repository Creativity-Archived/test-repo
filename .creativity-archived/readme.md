In [the config file](config.json), please make `type` and `whatType` a one word. For example:

**Example 1:**
```json
{
  "type": "Mods",
  "whatType": "Friday Night Funkin'"
}
```

**Example 2:**
```json
{
  "type": "Website",
  "whatType": "Sharing Games"
}
```

**Example 3:**
```json
{
  "type": "Software",
  "whatType": "Utility Software"
}
```
