# TEST-REPO
This repo is a test for a new feature being added where it would read [.creativity-archived](.creativity-archived) folder for [banner.png](.creativity-archived/banner.png), [config.json](.creativity-archived/config.json), [logo.png](.creativity-archived/logo.png) and [readme.md](.creativity-archived/readme.md).

## Configuration? ([config.json](.creativity-archived/config.json))
- `isDownloadable` > If it's downloadable, the download button would show up.
- `isOpenSource` > If it's open source, the tag will show if it is or not. (yes/no)
- `canMessWithComputer` > If it messes with your computer, it will show if it is or not. (yes/no)
- `license` > What license your project uses
- `NSFW` > Is your project NSFW? (yes/no)
>
- `type` > What type is your project? A Website? Software? Mod? Game?
- `whatType` > And what type is it? A mod for FNF? A software for utility?
>
- `hasGithub` > Does your project have a github? (yes/no)
- `hasWebsite` > Does your project have a website? (yes/no)
- `hasDiscord` > Does your project have a discord? (yes/no)
>
- `title` > What the title should be for your project. (eg: **Creativity Archived**)
- `description` > What the description should be for your project. Shown in the main page and not the popup.
- `madeBy` > Who made it? A team of people? or a single person? (eg: **Creativity Archived Team** or **Daveberry Cricket Blueson**)
>
- `githubURL` > What your GitHub URL will redirect to. You can put "none" if you put no in **has**.
- `discordURL` > What your Discord URL will redirect to. You can put "none" if you put no in **has**.
- `websiteURL` > What your Website URL will redirect to. You can put "none" if you put no in **has**.
- `madeByURL` > The team/person URL who made the project. Linked to their website.
